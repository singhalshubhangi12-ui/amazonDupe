# amazonDupe
# Amazon Clone

A simple Amazon-inspired e-commerce website built using **Node.js, Express.js, EJS, MongoDB, HTML, CSS, and JavaScript**.

## 🚀 Features

- 🔐 User Signup & Login
- 🛍️ Electronics Products Page
- 🛒 Add to Cart
- ❌ Remove from Cart
- 🔍 Search Products
- 📦 Returns Page
- 🌐 Redirect to Official Amazon Website
- 🗄️ MongoDB Database Integration

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- EJS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose
